# AI.Web Slice 6 Authority Scanner Scaffold

This scaffold adds a deterministic local scanner for boundary terms and overreach claims.

It is additive only.

It does not change dependencies.
It does not change runtime routes.
It does not change UI.
It does not write memory, evidence, corpus, external-resource, delivery, or action records.
It does not replace retained baselines.
It does not revive failed paths.
It does not grant release, readiness, or public-success status.

Negative examples are rebuilt from fragments during tests.
That keeps the scanner strict without storing blocked phrases as contiguous source text.

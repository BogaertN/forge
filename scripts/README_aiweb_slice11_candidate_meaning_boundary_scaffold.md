# AI.Web Slice 11 Candidate Meaning Construction Boundary Scaffold

Slice 11 adds deterministic boundary records for candidate meaning construction.

It may represent:

- source expression custody;
- derived structure custody;
- source-grounded candidate meaning records;
- upstream Slice 7, Slice 8, Slice 9 and Slice 10 boundary references;
- competing candidate separation;
- missing support;
- no-action safe candidate states.

It must not perform final meaning selection, truth decision, action authorization,
tool invocation, delivery, memory write, evidence validation, external resource
admission, expression rendering, release, production readiness, or live runtime
interpretation.

Frozen legacy files such as `rmc_engine_v1/llm_renderer.py` remain watchdog
inputs only. Slice 11 must not import them, call them, modify them, or rely on
any model-mediated authority.

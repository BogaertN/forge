# Patch 258B — AI.Web OS Desktop Launcher Icon

This patch installs a generated AI.Web OS desktop launcher image and creates a desktop shortcut.

It does not modify Forge `main.py`, does not add Forge commands, and does not bypass Forge.

Installed user-level files after running the installer:

- `~/.local/share/icons/aiweb/aiweb-os-desktop-launcher.png`
- `~/.local/share/applications/aiweb-os-desktop-launcher.desktop`
- `~/Desktop/AI.Web OS Desktop Launcher.desktop`

If Ubuntu asks, right-click the desktop icon and choose **Allow Launching**.
